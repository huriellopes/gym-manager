# Gym Manager

> Gerenciador de Academias do curso LaunchBase, feita com o intuito de cadastros de professores e membros da academia!

### Tecnologias
  - NodeJs
  - Express
  - Nunjucks
  - Moment
  - npm-run-all
  - nodemon
  - browser-sync
  - method-override
